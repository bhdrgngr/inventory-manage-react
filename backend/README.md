# Backend Projesi

Bu proje, FastAPI kullanarak geliştirilen bir backend uygulamasıdır.

## Kurulum

Projeyi Git'ten klonladıktan sonra:

1. **Bağımlılıkları yükleyin**:
   ```bash
   uv sync
   ```

2. **Uygulamayı çalıştırın**:
   ```bash
   uv run python main.py
   ```

## API Dokümantasyonu

Uygulama çalıştırıldığında, API dokümantasyonuna http://localhost:8000/docs adresinden erişebilirsiniz.
